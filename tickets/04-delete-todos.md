# Todo 削除 API の実装

## 概要
既存の Todo を削除する `DELETE /todos/:id` エンドポイントを実装してください。

## エンドポイント仕様

### `DELETE /todos/:id`

#### パスパラメータ
| パラメータ | 型 | 説明 |
|---|---|---|
| id | number | 削除対象の Todo ID |

#### レスポンス

- **204 No Content** — 削除成功 (レスポンスボディなし)
- **404 Not Found** — `{ "error": "Todo not found" }`

## 実装上の注意
- インメモリ配列 `todos` から該当要素を削除してください。
- 204 レスポンスではボディを返さないでください。

## 受け入れ基準
- [ ] 存在する Todo を削除でき 204 が返る
- [ ] 削除後に `GET /todos/:id` すると 404 が返る
- [ ] 存在しない ID で 404 が返る
