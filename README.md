# GitHub Copilot Agent 研修

Todo 管理 REST API を GitHub Copilot Agent を使って実装する研修です。

## セットアップ

```bash
npm install
```

## 開発

```bash
# 開発サーバ起動 (ホットリロード)
npm run dev

# ビルド
npm run build
```

## 課題の進め方

1. [チケット一覧](https://github.com/aYukiSekiguchi/test-agent-training/issues) から課題を選ぶ (番号順に進めてください)
2. チケットの内容を Copilot Agent に読み込ませて実装する
3. 受入試験を実行して合格を確認する

## テスト

```bash
# 自分で書いたユニットテストの実行
npm test

# 受入試験の実行
npm run test:submit
```
